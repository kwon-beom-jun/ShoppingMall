# 🛒 Shopping Mall

현 프로젝트를 진행하며 알고만 있던 기술들을 적용해 개발 역량을 향상시키기 위한 프로젝트

<br>

## 🖥 Project Development Environment
- IDE : Intellij
- JDK : 11
- Build Tool : Maven
- SpringBoot : 2.5.2
- Database : MySQL
- JPA :
- Server : Embedded Tomcat

<br><br>

## 📃 프로젝트 기능
> 1. 상품등록
> 2. 상품 관리
> 3. 장바구니
> 4. 구매이력
> 5. 로그인/로그아웃

<br><br>

### 🔥 목표
> JPA, Spring Security를 사용하여 프로젝트 개발

<br><br><br>
